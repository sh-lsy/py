# Python 学习之旅

欢迎来到 Python 学习仓库！这里记录了 Python 编程语言的学习过程、代码示例和相关笔记。

## 仓库结构

本仓库将包含以下内容：

- `base/`: 基础语法练习与入门代码
- 数据结构与算法
- 常用标准库示例
- 第三方库使用探索
- 小项目实战

## 环境设置

建议使用 Python 3.8+ 版本。

### 安装依赖

```bash
pip install -r requirements.txt
```
